# ربات نیم بها کننده لینک تلگرام

یک ربات تلگرام که فایل‌های شما را در آپلود سنترهای ایرانی آپلود می‌کند و لینک دانلود نیم بها دریافت می‌کنید.

## ✨ امکانات

- 📁 آپلود فایل‌های فوروارد شده از تلگرام
- 🔗 آپلود از لینک‌های مستقیم
- 🌐 پشتیبانی از آپلود سنترهای مختلف
- 🗑️ حذف خودکار فایل‌ها پس از آپلود
- ⚡ پشتیبانی از فایل‌های تا 2 گیگابایت

## 🚀 نصب و راه‌اندازی

### 1. دریافت توکن ربات

1. به @BotFather در تلگرام بروید
2. دستور `/newbot` را ارسال کنید
3. نام و یوزرنیم ربات را وارد کنید
4. توکن دریافتی را ذخیره کنید

### 2. کلون کردن پروژه

```bash
git clone https://github.com/your-username/nim-behah-bot.git
cd nim-behah-bot
```

### 3. تنظیم متغیرهای محیطی

```bash
cp .env.example .env
```

فایل `.env` را ویرایش کنید و توکن ربات را وارد کنید:

```
BOT_TOKEN=your_token_here
```

### 4. نصب وابستگی‌ها

```bash
pip install -r requirements.txt
```

### 5. اجرای ربات

```bash
python bot.py
```

## 🐳 اجرا با Docker

### با Docker Compose

```bash
docker-compose up -d
```

### با Docker

```bash
docker build -t nim-behah-bot .
docker run -d --name nim-behah-bot --env-file .env nim-behah-bot
```

## 📱 استفاده از ربات

1. ربات را در تلگرام پیدا کنید و `/start` بزنید
2. فایل مورد نظر را به ربات فوروارد کنید
3. یا لینک مستقیم دانلود فایل را ارسال کنید
4. لینک نیم بها دریافت کنید!

## ⚙️ دستورات ربات

| دستور | توضیح |
|-------|-------|
| `/start` | شروع مجدد ربات |
| `/help` | راهنمای ربات |
| `/list` | لیست آپلودرهای موجود |
| `/set_uploader` | تنظیم آپلودر پیش‌فرض |

## 🌐 استقرار روی هاست رایگان

### Render.com

1. یک حساب کاربری در [Render.com](https://render.com) بسازید
2. یک **Background Worker** جدید ایجاد کنید
3. ریپازیتوری گیت‌هاب خود را متصل کنید
4. متغیرهای محیطی را تنظیم کنید:
   - `BOT_TOKEN`: توکن ربات
5. Deploy را بزنید

### Railway.app

1. یک حساب کاربری در [Railway.app](https://railway.app) بسازید
2. یک پروژه جدید ایجاد کنید
3. ریپازیتوری گیت‌هاب را متصل کنید
4. متغیرهای محیطی را تنظیم کنید
5. Deploy کنید

### Koyeb.com

1. یک حساب کاربری در [Koyeb.com](https://koyeb.com) بسازید
2. یک سرویس جدید با **Docker** ایجاد کنید
3. ایمیل Docker Hub خود را وارد کنید
4. متغیرهای محیطی را تنظیم کنید
5. Deploy کنید

## 📝 نکات مهم

- **حجم فایل**: حداکثر حجم فایل 2 گیگابایت است (قابل تغییر)
- **پاکسازی**: فایل‌ها پس از آپلود به صورت خودکار حذف می‌شوند
- **آپلودرها**: در صورت خرابی یک آپلودر، ربات به صورت خودکار از آپلودرهای دیگر استفاده می‌کند
