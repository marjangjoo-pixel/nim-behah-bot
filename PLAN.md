# 🛍️ فروشگاه آنلاین ایرانی - برنامه پروژه

## هدف
ساخت یک فروشگاه آنلاین مدرن ایرانی مشابه دیجی‌کالا/ترب با کمترین هزینه ممکن.

## استک فناوری (رایگان)

| لایه | فناوری | هزینه |
|------|--------|-------|
| فرانت‌اند | Next.js 14 + Tailwind CSS | رایگان |
| بک‌اند | Next.js API Routes | رایگان |
| دیتابیس | Supabase (PostgreSQL) | رایگان (50K rows) |
| فایل/عکس | Supabase Storage | رایگان (1GB) |
| احراز هویت | Supabase Auth | رایگان |
| پرداخت | زرین‌پال | رایگان (کارمزد تراکنش) |
| هاست | Vercel | رایگان |
| دامنه | .ir | ~50K تومان/سال |

## صفحات اصلی

### ۱. صفحه اصلی (/)
- اسلایدر بنر تبلیغاتی
- دسته‌بندی‌های محبوب (آیکون + نام)
- محصولات ویژه / پرفروش
- بنر تخفیف ویژه
- برندهای محبوب

### ۲. لیست محصولات (/products)
- نمایش شبکه‌ای محصولات
- فیلتر: دسته‌بندی، قیمت، برند
- مرتب‌سازی: جدیدترین، ارزان‌ترین، گران‌ترین
- جستجوی سریع
- صفحه‌بندی

### ۳. جزئیات محصول (/products/[id])
- گالری عکس
- اطلاعات کامل محصول
- انتخاب رنگ/سایز
- دکمه افزودن به سبد
- نظرات کاربران
- محصولات مشابه

### ۴. سبد خرید (/cart)
- لیست اقلام
- تغییر تعداد
- حذف آیتم
- محاسبه جمع کل
- دکمه ادامه فرآیند خرید

### ۵. پرداخت (/checkout)
- فرم اطلاعات ارسال
- انتخاب روش ارسال
- خلاصه سفارش
- اتصال به درگاه زرین‌پال

### ۶. پنل مدیریت (/admin)
- داشبورد آماری
- مدیریت محصولات (CRUD)
- مدیریت سفارشات
- مدیریت دسته‌بندی‌ها

## دیتابیس (Supabase)

### جداول اصلی
1. **products** - محصولات
2. **categories** - دسته‌بندی‌ها
3. **orders** - سفارشات
4. **order_items** - اقلام سفارش
5. **reviews** - نظرات
6. **users** - کاربران (از Supabase Auth)
